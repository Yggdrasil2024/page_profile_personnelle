# 📋 Suivi de Projet : Carte Produit Stylisée (Projet 3.2)

> **Durée estimée :** 3-4 heures ⏳
> **Objectif principal :** Prendre le HTML du projet précédent (S2.1) et le styliser entièrement en CSS pur.

## 🗂️ 1. Configuration & Livrables Initiaux
- [ ] Créer le fichier `style.css` dans le bon dossier.
- [ ] Modifier le fichier HTML existant (`index.html`) pour lier la feuille de style externe.
- [ ] Vérifier la liaison en appliquant un style temporaire (ex: un fond rouge sur le body).

## 🎨 2. Éléments CSS à Intégrer
- [ ] **Couleurs :** Définir et appliquer les couleurs de fond (`background-color`) et de texte (`color`).
- [ ] **Typographie :** Importer et appliquer au moins 2 polices **Google Fonts**.
- [ ] **Bordures :** Ajouter des bordures simples (`border`, `border-radius` pour adoucir si besoin).
- [ ] **Espacements :** Configurer les marges internes (`padding`) et externes (`margin`) pour aérer la carte.
- [ ] **Box Model :** Appliquer le Box Model de manière globale et visible sur tous les éléments.

## ✅ 3. Critères d'Évaluation (Éliminatoires / Obligatoires)
- [ ] **Zéro style inline :** Tout le CSS est dans `style.css` (aucune balise `<style>` ou attribut `style=""` dans le HTML).
- [ ] **Variété des sélecteurs :** Utiliser au moins **10 sélecteurs CSS différents** (sélecteurs de balise, de classe, d'identifiant, ou combinés).
- [ ] **Maîtrise du Box Model :** Le comportement des boîtes est logique, maîtrisé et visible à l'écran.
- [ ] **Lisibilité et Esthétique :** La page finale est attrayante, propre et agréable à lire.
- [ ] **Deux polices minimum :** Utilisation correcte d'au moins 2 Google Fonts distinctes (ex: une pour les titres, une pour le texte).

## 💡 4. Conseils & Bonnes Pratiques (À valider)
- [ ] **Palette restreinte :** Limiter le design à **3 couleurs maximum** pour garantir une harmonie visuelle.
- [ ] **Règle des 8px :** Assurer des espacements cohérents en utilisant uniquement des multiples de 8px (ex: `8px`, `16px`, `24px`, `32px`).
- [ ] **Choix des polices :** Sélectionner des polices hautement lisibles (comme *Roboto*, *Open Sans*, *Inter*, etc.).
